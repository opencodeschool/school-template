This is a second example lesson showing that lessons can have different structures while following the same frontmatter format.

## Overview

Introduce the lesson topic and explain why students should care about it.

## Core concepts

Break down the key ideas. Use clear language and build on what students already know from previous lessons.

## Real-world examples

Concrete examples make abstract concepts stick. Show how the material applies in practice.

## Summary

Connect this lesson to the broader course. What did students just learn, and how does it prepare them for the next lesson?
